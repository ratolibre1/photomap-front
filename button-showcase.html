Catálogo de Botones - PhotoMap

Este documento muestra visualmente todos los diferentes tipos de botones utilizados en la aplicación PhotoMap. ¡Prueba a cambiar el tema con el selector a la derecha!

Nota: Los botones de este catálogo son solo de muestra y no responden a interacciones. Los estados hover y active están simulados visualmente para propósitos de documentación.

1. Botones Principales (Primary)

Botón Principal Sólido (btn-primary)

Botón principal para acciones primarias o llamadas a la acción destacadas.

Normal
Fondo: var(--primary) | Texto: Blanco | Borde: var(--primary)
Hover
Fondo: var(--btn-primary-hover) | Transformación: scale(1.03) | Sombra aumentada
Activo/Presionado
Transformación: scale(0.98)
Inactivo/Deshabilitado
Fondo: #999 | Texto: Blanco | Opacidad: 100%

Código:

<Button variant="primary">Iniciar Sesión</Button>
<Button variant="primary" size="lg">Crear Mapa</Button>
<Button variant="primary" disabled>Iniciar Sesión</Button>

Dónde se utiliza:

  • Login/Registro: Botones de "Iniciar Sesión" y "Registrarse" en las pantallas de autenticación
  • Dashboard: Botón para crear nuevo mapa
  • Subida de Fotos: Botón de confirmación para finalizar la subida
  • Configuración de Cuenta: Botón para guardar cambios importantes en perfil
  • Formularios: Como botón de envío (submit) en formularios principales
  • Modales: Para acciones principales en ventanas modales

Botón Principal Outline (btn-outline-primary)

Variante de contorno del botón principal, para acciones importantes pero secundarias.

Normal
Fondo: Transparente | Texto: var(--btn-primary-bg) | Borde: var(--btn-primary-bg)
Hover
Fondo: var(--btn-primary-bg) | Texto: Blanco | Transformación: scale(1.03)
Inactivo/Deshabilitado
Fondo: #999 | Texto: Blanco

Código:

<Button variant="outline-primary">Ver Perfil</Button>
<Button as={Link} to="/gallery" variant="outline-primary">{t('gallery.button')}</Button>

Dónde se utiliza:

  • Dashboard: Enlaces a diferentes secciones de la aplicación
  • Perfil de Usuario: Opciones alternativas como "Ver Galería" o "Ver Mapas"
  • Navegación Secundaria: En barras de navegación como opciones secundarias
  • Filtros: Para aplicar filtros en galerías y mapas
  • Modales: Como alternativa secundaria en diálogos (p.ej., "Ver detalles")
  • Listas: Botones de acción secundarios en listas de elementos

9. Guía General de Uso

Recomendaciones generales para la consistencia en el uso de botones en toda la aplicación

  • Primary: Usar sólo para la acción principal en cada pantalla o sección. Siempre con texto en negrita.
  • Success: Para acciones que confirman, completan o crean elementos.
  • Danger: Exclusivamente para acciones destructivas que requieren consideración.
  • Secondary: Para acciones de cancelación o alternativas secundarias.
  • Action: Para acciones contextuales específicas en elementos individuales.
  • Estado Inactivo: Todos los botones deshabilitados usan el mismo estilo: fondo gris #999 con texto blanco, sin importar el tipo de botón.
  • Tamaños: Usar consistentemente según importancia - lg para acciones destacadas, sm para acciones compactas.
